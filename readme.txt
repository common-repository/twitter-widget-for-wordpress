== Description ==

The Twitter Widget for WordPress-project is a small(ish) and simple WordPress widget for retrieving, caching and displaying tweets from one specific Twitter user. It is build upon the Twitter Widget library. The entire library is built with an object oriented approach which makes it very flexible and extendable.

== Installation ==

1. Upload the 'twitter-widget-for-wordpress' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the widget to your sidebar on the 'Widget' menu in WordPress.
